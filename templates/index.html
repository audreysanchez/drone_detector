Drone Detector

The purpose of this web app is to demonstrate the results of the trained convoluted neural network(CNN) by choosing a random image from a folder that has drone and not drone images.
Then the random image chosen will be run through the program and the result will be displayed.
Right now the web app was made so that you have to reload the page so a new random image is chosen so please be patient and wait for the page to load.
